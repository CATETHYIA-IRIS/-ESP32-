灵汐 Android App 工程说明

版本：1.0.1
App 名称：灵汐

本版本调整：
1. App 名称改为“灵汐”。
2. 启动页只显示“灵汐全屋智能控制中心”。
3. “本地控制”改为“本地访问”。
4. “远程控制”改为“远程访问”。
5. 删除启动页提示说明和 WebView 套壳相关文字。
6. 使用用户提供的灵汐 Logo 作为 App 图标和启动页 Logo。

使用方法：
1. 解压工程压缩包。
2. 用 Android Studio 打开外层 Lingxi_Android_WebView_App_V1_0_1_CleanUI 文件夹。
3. 等待 Gradle Sync 完成。
4. 连接手机运行，或 Build -> Generate App Bundles or APKs -> Generate APKs 生成 APK。

注意：
工程路径建议放在纯英文目录，例如 C:\LingxiApp\，避免中文路径导致 Gradle 构建失败。
